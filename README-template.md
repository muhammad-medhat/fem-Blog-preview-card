## My process

I created a blog preview card using HTML and CSS. I structured the content with semantic HTML elements and styled it to match the provided design. I focused on making the layout responsive and ensuring that interactive elements had appropriate hover states for better user experience.

### Built with

- Semantic HTML5 markup
- CSS custom properties
- Flexbox
- CSS Grid
- Mobile-first workflow

### What I learned

During this project I focused on my development practices, keeping components modular and reusable while making the layout responsive and accessible. I learned how to structure CSS to support the card preview design and how to combine semantic HTML with clean styles for a polished result.
